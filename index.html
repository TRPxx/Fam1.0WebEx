แจ้งเตือน
ยินดีต้อนรับสู่เว็บไซต์ Fam 1.0
โปรดอ่านเงื่อนไขและรายละเอียดการเทรดของให้ครบถ้วนก่อนดำเนินการ
หากมีข้อสงสัยสามารถติดต่อทีมงานได้ตลอดเวลา
ผ่านทาง Ticket Server Fam 1.0
เงื่อนไขการเทรดของ สถานที่การเทรดของ รายละเอียดเหรียญ วิธีการเทรดของ การแลกเปลี่ยนอาวุธ
เงื่อนไขการเทรดของ FAM 1.0
ไอเทมทั่วไป
- ไอเทมประเภททั่วไปที่ไม่ใช่อาวุธหรือแฟชั่น จะถูกแปลงมูลค่าเป็นเงินสดและนำมาคำนวณเพื่อแลกเป็น "Ticket Money" โดยกำหนดว่า ทุก ๆ 500,000 หรือ 1,000,000 หน่วยเงินที่แปลงได้ จะสามารถแลกเป็น 1 Ticket Money ทั้งนี้ เศษเงินที่ไม่ครบตามเรตที่กำหนดจะไม่ถูกนำมาคิดรวม
Weapon (จากเซิร์ฟเวอร์ Fam 1.0)
- จะได้รับการปรับเปลี่ยนระดับให้เทียบเท่ากับระบบ Tier อาวุธที่ใช้งานในเซิร์ฟเวอร์ Fam Unity เพื่อให้เกิดความเหมาะสมและสมดุลในการใช้งานร่วมกันภายใต้ระบบใหม่
* หากผู้เล่นมีข้อมูลทั้ง Server Fam 1.0 และ Fam Freedom สามารถเลือกการเทรดอาวุธได้เพียงเชิร์ฟเวอร์เดียวเท่านั้น
** อาวุธแต่ละชนิดสามารถพกไปได้ชิ้นเดียวเท่านั้น **
แฟชั่น
- แฟชั่นทุกประเภท จะถูกแปลงเป็น “เหรียญแฟชั่น” ซึ่งสามารถนำไปแลกเปลี่ยนเป็นแฟชั่นอื่น ๆ ภายใน Fam Unity ได้ในอนาคต
รถจากกิจกรรมประมูล
- รถที่ได้รับจากกิจกรรมประมูลจะถูกแปลงเป็น “เหรียญรถประมูล” โดยการแปลงจะอ้างอิงจากผู้ถือกุญแจคนล่าสุดของรถนั้น
รถจากระบบบริจาค (Top Donate)
- รถที่ได้รับจากการบริจาคในระดับ Top Donate จะถูกแปลงเป็น “เหรียญรถ Top Donate” ซึ่งสามารถนำไปใช้ในกิจกรรมหรือร้านค้าเฉพาะภายใน Fam Unity ได้
รถประกอบ
- รถที่ผู้เล่นประกอบขึ้นเองในเซิร์ฟเวอร์เดิมจะถูกแปลงเป็น “เหรียญรถประกอบ”
รถกาชา
- รถที่ได้รับจากระบบสุ่มกาชาจะถูกแปลงเป็น “เหรียญรถกาชา”
แฟชั่นจากระบบบริจาค (Top Donate)
- แฟชั่นที่ได้รับจากการบริจาคพิเศษจะถูกแปลงเป็น “เหรียญแฟชั่น Top Donate”
สกินอาวุธ
- สกินตกแต่งอาวุธทั้งหมดจะถูกแปลงเป็น “เหรียญสกินอาวุธ”
